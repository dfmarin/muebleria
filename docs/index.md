# Bienvenido a la documentación de la Mueblería

## Aplicación

Esta aplicación es un sistema orientado a objetos desarrollado en Python para la gestión de una mueblería.
Permite administrar el inventario de muebles, realizar ventas y registrar clientes de forma sencilla.

El diseño se basa en clases como Mueble, Silla, Escritorio, Cliente y Muebleria, lo que facilita la extensibilidad y el mantenimiento del código.

Cada clase encapsula su propia lógica de negocio, y el sistema está estructurado de forma que puede crecer fácilmente para incluir nuevas funcionalidades, como gestión de proveedores, facturación o persistencia en bases de datos o archivos.

Este proyecto es ideal como ejemplo práctico para introducir los principios de programación orientada a objetos (POO) en Python, mostrando herencia, encapsulación y polimorfismo en un contexto realista.
